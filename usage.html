Features

  • Placing and Moving Dots

    1. Click in the bounds of the field to place a dot. It will be automatically selected.
    2. Click on an existing dot to select it.
    3. Click and drag a dot to move it.
    4. When clicking and dragging, right-clicked to cancel the move.
    5. Right-click a dot to delete it.
    6. Right click any empty space to deselect the currently selected dot.

  • Editing Dots

    The “Dot Data” pane on the right side of the screen contains controls for the selected dot.

    Controls:

    1. Instrument
    2. When clicking and dragging, right-clicked to cancel the move.
    3. Number
    4. When clicking and dragging, right-clicked to cancel the move.
    5. X Position
    6. When clicking and dragging, right-clicked to cancel the move.
    7. Y Position
    8. When clicking and dragging, right-clicked to cancel the move.
    9. Position Text
    10. When clicking and dragging, right-clicked to cancel the move.

    Notes:

    1. The instrument and number make the dot “name”. Duplicate names are allowed technically, although it doesn’t make much sense from a marching standpoint. Telling someone to be in two places at once is a bit evil, even for a band director.

  • Editing the Page

    The “Page Data” pane on the right side of the screen contains controls for the entire page.

    1. Navigation
    2. Pages are shown in the format:
    3. [Page Number] | [Song Title], m.[Starting Measure]-[Ending Measure]
    4. - Select a page to display that page. You don’t need to save the page you’re working on before you do this, but it never hurts.
    5. Page Number
    6. Just shows the page number… You didn’t really have a question about this one, right?
    7. Song Title
    8. Edit the song title here and watch as it magically updates the navigation box and text box.
    9. Starting Measure
    10. ^
    11. Ending Measure
    12. ^
    13. Counts
    14. ^
    15. Notes
    16. Write whatever you want. It will be shown in the text box.
    17. Text X Position
    18. Move the text box horizontally.
    19. Text Y Position
    20. Move the text box vertically.
    21. Clear Page
    22. Remove all dots from the page. Does not save automatically, so if you somehow managed to do this accidentally, just close without saving and re-open.
    23. Delete Page
    24. Remove this page and change the number of other pages to remove the hole. Band directors hate holes. Does not save automatically, so if you somehow managed to do this accidentally, just close without saving and re-open.

  • Display Settings

    1. Toggle Gridlines
    2. Click it once and the grid disappears. Click it again and it’s back. Abracadabra!
    3. Toggle Dot Names
    4. Clap on. Dot names disappear. Clap off. Dot names re-appear.
    5. Undo
    6. Remember that thing you just did? Because I don’t.

  • File Menu

    1. New
    2. Create a new show in the DrillSweet directory by default or wherever you choose. It will ask you for a filename for the new page. You don’t need to give it the “.ds2” extension.
    3. Open
    4. Open an existing show in the DrillSweet directory by default or wherever you choose.
    5. Save
    6. Save the current show under the current filename. The current filename is displayed in the title bar and taskbar next to “DrillSweet 2”.
    7. Save as
    8. Save the current show under a new filename.
    9. Print Current Page
    10. Saves the current show as a PDF file in the DrillSweet directory. Takes current “toggle gridlines” and “toggle dot names” states into account.
    11. Print Show
    12. Saves every page as a PDF file. The name of the PDF file will be the same as the name of the file of the show, plus “full show”.
    13. Print Dot Sheets
    14. Saves dot sheets for every player as PDF files named by their instrument letter. These are all put in a folder called the name of your show “Dot Sheets”. Hope you got all that.
    15. Quit
    16. Now why would you want to do that?

  • Click Controls

    TL;DR:

    1. Shift-click to draw a rectangle and select all dots in it. Right-click to remove all selected dots.
    2. Control-click to select/deselect multiple dots one by one. Right-click to remove all selected dots.
    3. Alt-click to select/deselect all dots by instrument Right click to remove all selected dots.
    4. Normal click to add or select a dot. Drag to move selected dot(s). Right-click to remove dot.

      5. Full Details:

      Holding Shift:
      1. Before Drag
      2. Right click on selected point to remove all selected points
      3. Right click on deselected point to do NOTHING
      4. Right click off point to deselect all
      5. Left-click (on or off point) to drag
      6. During drag:
      7. Draw and show rectangle
      8. Right click (on or off point) during drag to cancel
      9. Release left click (on or off point) to stop drag and select all in rectangle
      Holding control:
      1. Left click on selected point to deselect point
      2. Left click on deselected point to select point
      3. Left click off point to do NOTHING
      4. Right click on selected point to remove all selected points
      5. Right click on deselected point to deselect all
      6. Right click off point to deselect all
      Holding alt:
      1. Left click on selected point to deselect all points with same instrument
      2. Left click on deselected point to select all points with same instrument
      3. Left click off point to do NOTHING
      4. Right click on point to delete all points with same instrument
      5. Right click off point to deselect all
      Normal:
      1. Before drag:
      2. Left click off point to add and select point
      3. Left click on deselected point to select point
      4. Right click on selected point to deselect and remove all selected points
      5. Right click on deselected point to remove point
      6. Right click off point to deselect selected point
      7. Left click on selected point to drag
      8. During drag:
      9. Move and show selected point(s)
      10. Right click on or off point during drag to cancel
      11. Release left click (on or off point) to stop drag and deposit selected point(s)

Terminology

  1. Page
  2. Individual sheet of drill paper. Also known as a set. A page has a set of information associated with it.
  3. Show
  4. Set of pages. An entire show is stored in one DS2 file which can be opened as a JSON file.
  5. Dot
  6. Individual person represented on a page. A dot has a a set of information associated with it.
  7. DrillSweet directory
  8. Documents/DrillSweet 2/ by default
